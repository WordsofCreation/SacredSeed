# Logo upload drop zone

Place all original SacredSeed logo files in this folder.

Recommended filenames:

- `sacredseed-logo-primary.png`
- `sacredseed-logo-mark.png`
- `sacredseed-logo-wordmark.png`
- `sacredseed-logo-light.png`
- `sacredseed-logo-dark.png`

Notes:

- Keep originals at the highest available resolution.
- Do not manually resize before upload; variants can be generated later with `python3 scripts/process_images.py`.
- Prefer PNG or SVG originals when possible.
